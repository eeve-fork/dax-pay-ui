# DaxPay WEB端

## 简绍
vite5 + vue3.4 + vben admin2 + antd vue4 搭建

## 系统环境要求

| 环境   | 版本        | 备注                       |
|------|-----------|--------------------------|
| Node | >=18.12.0 | 推荐使用20.9.0+              |
| pnpm | >=9.0.2+	 | 低版本会导致 pnpm-lock.yaml 失效 |

## 项目启动
```shell
# 安装项目依赖
pnpm install
# 启动项目
pnpm run dev
# 构建打包
pnpm build
```
